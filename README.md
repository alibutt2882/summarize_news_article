# summarize_news_article
AI Article Summarizer using NLP and Hugging Face Transformers
Developed an intelligent article summarization system using Natural Language Processing (NLP) and Hugging Face Transformers. The system utilizes the BART (facebook/bart-large-cnn) pretrained model to perform abstractive text summarization, generating concise summaries from long articles or web content. The application can extract text from URLs using web scraping (BeautifulSoup) or accept direct user input, then process it through a deep learning transformer model to produce meaningful summaries. A Streamlit-based interactive interface allows users to easily input text and view AI-generated summaries in real time.

Technologies: Python, NLP, Hugging Face Transformers, BART Model, Streamlit, BeautifulSoup, PyTorch.
